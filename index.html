What is SPSS?

SPSS = Statistical software for data analysis
Used extensively for surveys and academic research
Transforms raw data into actionable insights

SPSS Interface

Data View
Displays the data matrix where each row is a case and each column is a variable.
Variable View
Defines variable properties such as name, type, and label.

Data Structure

Rows = observations (respondents)
Columns = variables
Cells = values

Types of Variables

Nominal → categories (e.g., gender)
Ordinal → ranking (e.g., satisfaction level)
Interval → numeric (e.g., age)
Type Description Example
Nominal Categories with no inherent order Gender, Eye Color
Ordinal Ranked categories with meaningful order Satisfaction (1–5), Education Level
Interval Numeric values with equal intervals Age, Temperature, Income

Variable Definition

Name
Short label with no spaces
(e.g., age, gender)
Type
Defines data format
(e.g., numeric or string)
Label
Full descriptive name of the variable for clear output

Value Labels

Assign meaning to numeric codes
Example:
0 = Female
1 = Male

Missing Values

Missing data must be explicitly defined in SPSS
Example:
99 = no answer / not applicable
Properly defining missing values avoids incorrect analysis results

Data Entry

Enter data directly in Data View
Each row represents one respondent

Compute Variable

Create new variables based on existing ones
Example:
BMI calculation from weight and height

Select Cases

Filter data to analyze a specific subgroup
Example:
Analyze data for males only

Recode Variables

Transform existing variable values into new ones
Example: Reverse a scale
1 → 5
2 → 4
3 → 3

Why Descriptive Statistics?

Summarize large datasets into understandable figures
Identify patterns and trends in the data
Detect errors or anomalies before deeper analysis

Frequencies

Shows counts and percentages for each category
Particularly useful for nominal and ordinal (categorical) data

Frequency Output

Output table shows the distribution of responses
Helps with straightforward interpretation of categorical data

Charts in SPSS

Bar chart → comparison between categories
Pie chart → proportions of a whole
Histogram → distribution of continuous data

Descriptive Statistics

Mean Average value of the dataset
Median Middle value when data is sorted
Std. Deviation Measure of variability or spread

Descriptive Output

Output table shows a summary of key statistics
Helps understand the spread and central tendency of the data

Multiple Response Sets

Used when a question allows multiple answers (e.g., "Which brands do you use?")
Requires encoding each answer option as a separate variable

Multiple Response in SPSS

Define a multiple response set in SPSS
Analyze frequencies across all response variables

Interpretation of Results

Results must always be interpreted in context
Example
“60% of respondents prefer Brand A”
Link findings to practical decision-making

Summary

SPSS helps organize and structure data efficiently
Descriptive statistics summarize and describe the dataset
Interpretation of results is the key to meaningful insights
