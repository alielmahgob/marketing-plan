Chapter 1

Working with SPSS

Preparing and managing data in SPSS

Opening and Starting SPSS

1
SPSS can be started from the desktop or application menu
2
Users can begin with a new dataset or open an existing file
3
The starting dialog gives different options to begin working

Saving SPSS Files

Use File → Save or Save As
Choose location and file name carefully
Saving regularly is important during data work

The Two Main Working Views

Data View
Data View is used for entering observations
Variable View
Variable View is used for defining variables
Both views are essential for building a dataset

Defining Variables

Each variable needs a name and data type
Numeric variables are usually preferred for analysis
Labels help make variables easier to understand
Decimals and widths can also be adjusted

Making Codes Understandable

Value labels give meaning to coded numbers
Example: 0 = female, 1 = male
Value labels improve readability in data and output

Working with Missing Data

Missing values can be system missing or user-defined
Examples: blank cells, -1, 99
Missing values must be defined correctly so SPSS does not treat them as real data

Data Entry Options

Data can be typed directly into Data View
Data can also be imported from other sources
Correct structure is important before analysis begins

Creating Additional Variables

New variables can be calculated with Transform → Compute Variable
Formulas allow data transformation
Example: calculating BMI from height and weight

Filtering Cases for Analysis

Select Cases allows analysis of only part of the dataset
Excluded observations are not deleted permanently
Useful for subgroup analysis such as males only or females only

Comparing Groups Automatically

Split File organizes output by groups
Allows separate results for categories such as gender
Useful when comparing groups without creating separate files

Changing Variable Coding

Recoding adjusts values for analysis
It is safer to recode into a different variable
Useful when reversing scales or regrouping categories

Question 2: If I'm hungry between meals, I will eat fruit more often than candy.

Question 3: I always like to add extra sugar to my dessert.

Chapter 2

Descriptive Statistics

Summarizing data using descriptive statistics

Understanding Distributions

Frequency tables show counts and percentages
Useful for exploring data structure
Helpful for data cleaning and checking impossible values

Running Frequencies in SPSS

Use Analyze → Descriptive Statistics → Frequencies
Select one or more variables
Output appears in the Output Viewer

Reading Frequency Tables

Frequency = number of observations
Percent includes missing values
Valid percent excludes missing values
Cumulative percent shows running totals

Visualizing Frequencies with Charts

Charts make it easier to see patterns
Bar charts are good for categorical data
Pie charts show proportions of a whole

Questions with More Than One Answer

  • Some survey questions allow multiple answers
  • This changes the data structure
  • Instead of one variable, several variables may be needed

What are Multiple Response Sets?

Some survey questions allow more than one answer
Example: "Which brands do you know?"
SPSS requires these to be grouped before analysis

Defining Multiple Response Sets

Use Analyze → Multiple Response → Define Variable Sets
Select the variables that belong to the same question
Specify the counted value (usually 1 for "yes")

Limited-Choice Multiple Response Structure

  • When only a limited number of answers are allowed, categories are coded differently
  • A smaller number of variables can store the responses
  • SPSS can also analyze these as a multiple response set

Mean and Variation in the Data

Mean
The mean is the average of all values
It is the most common measure of central tendency
It can be sensitive to extreme values (outliers)
Variation
Standard deviation shows how much values differ from the mean
Minimum and maximum show the range of the data

Running Descriptive Statistics

Use Analyze → Descriptive Statistics → Descriptives
Select interval or continuous variables
Options allow choosing specific measures like mean and standard deviation

Reading Descriptive Statistics Output

The output table shows N, minimum, maximum, mean, and standard deviation
The mean gives the average value
The standard deviation shows how much the data varies around the mean
